FindIT – Indoor Navigation Platform

Built a production-oriented navigation platform that improves wayfinding for indoor spaces using a secure full-stack architecture and maintainable service design.

Problem

Users struggled with indoor wayfinding in complex venues and lacked a responsive, low-friction way to locate rooms and services.

Architecture

  • React frontend with component-driven UI and API integration.
  • Spring Boot backend exposing REST APIs with role-aware access.
  • SQL data layer tuned for consistent query response and reliability.

Tradeoffs

Prioritized maintainability and testability over fast but brittle implementation. Chose clear service boundaries and stronger validation to support long-term scaling.

Outcomes

  • Delivered end-to-end route and location workflows.
  • Improved software quality through stronger testing practices.
  • Shipped production-ready APIs with clear team handoff paths.

Demo Proof

Live deployment available for direct review, including routing and location workflows.
