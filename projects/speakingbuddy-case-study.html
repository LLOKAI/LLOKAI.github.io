SpeakingBuddy

Built a guided communication practice application focused on structured speaking sessions, user confidence building, and accessible interaction flow.

Problem

Learners needed a low-pressure environment to practice communication skills with structure, repetition, and clear progression.

Architecture

  • Interactive front-end experience with step-by-step speaking activities.
  • Python-backed logic for guided prompts and session flow.
  • Simple data model to capture progress and support iteration.

Tradeoffs

Prioritized clear, supportive user flow and stable core interactions first; advanced personalization and analytics were scoped for future releases.

Outcomes

  • Delivered a practical prototype for guided speaking practice.
  • Created a maintainable baseline for feature expansion.
  • Improved UX consistency with structured interaction states.
